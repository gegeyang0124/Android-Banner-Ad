# Banner-Ad
主要功能，轮播图片、播放视频、一屏多个轮播（最多3个屏）; 每张图片的停留时长由后台回传时长控制；可同时播放多个轮播视频。
下载项目，本项目仅供参考，部分需要进行修改。代码没有。
